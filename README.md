#Introduction
This simple application will demonstrate the integration of some tools Meituan is using. Hello world to Meituan tech stack.

#Usage

1. Using embedded jetty to run the code:
`mvn clean jetty:run`

2. The go to page:
`http://localhost:8080/hello-spring/login`
